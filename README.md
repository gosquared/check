# Check

[![travis][1]][2]

Redis backed service for monitoring metric data streams against
pre-defined thresholds.

## Installation

Add this line to your application's Gemfile:

    gem 'check'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install check

## Usage

Check the examples directory.
